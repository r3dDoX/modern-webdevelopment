[ Arrays ] are Objects

An Array is basically a map with index-to-value pairs. This means there is no such thing as out of bounds

                    
                    const array = ['hello', 'array'];

                    const arrayish = {
                        0: 'hello',
                        1: 'array'
                    };

                    array[4] = 'I am not out of bounds!';

                    console.log(arrayish[0]); // hello
                    console.log(array[0]);    // hello
                    console.log(array[4]);    // I am not out of bounds!
                    console.log(array[5]);    // undefined

[ Arrays ] are Objects

When setting a value at an index that would leave gaps, those gaps are "filled".

                    
                    const array = [1, 2];
                    array[3] = 4;

                    console.log(array); // [1, 2, undefined, 4]

[ Array ] Operations

Adding a value

                    
                    const a = [1, 2];
                    a.push(3);

                    console.log(a); // [1, 2, 3]
push() mutates the array!

[ Array ] Operations

Removing last value

                    
                    const a = [1, 2, 3];

                    console.log(a.pop()); // 3
                    console.log(a); // [1, 2]
pop() mutates the array!

[ Array ] Operations

Concatenation

                    
                    const a = [1, 2];
                    const b = [3, 4];
                    const both = a.concat(b); // [1, 2, 3, 4]
concat() doesn't mutate the array but returns a new one

[ Array ] Operations

Splice

                    
                    const a = [1, 2, 3, 4];

                    const spliced = a.splice(2, 2);

                    console.log(a); // [1, 2]
                    console.log(spliced); // [3, 4]

                    spliced.splice(1, 0, 5, 5, 5);
                    console.log(spliced); // [3, 5, 5, 5, 4]
splice() mutates the array and returns the spliced elements

[ Array ] Operations

Slice

                    
                    const a = [1, 2, 3, 4];

                    const sliced = a.slice(2, 3);

                    console.log(a); // [1, 2, 3, 4]
                    console.log(sliced); // [3]
slice() does not mutate the array but returns a new one

[ Array ] Operations

Slice vs. Splice

Slice

  • arguments: fromIndex, toIndex
  • Returns new Array

Splice

  • arguments: fromIndex, deleteCount, itemsToInsert
  • Mutates the Array
  • Returns the deleted items as array

Iterating over an [ Array ]

Classical for-loop

                    
                    const arr = [1, 2, 3, 4];

                    for (let i = 0; i < arr.length; i++) {
                        console.log(arr[i]);
                    }

Iterating over an [ Array ]

for-of loop

                    
                        const arr = ['a', 'b', 'c'];

                        for (let value of arr) {
                            console.log(value);
                        }

                        // a
                        // b
                        // c
