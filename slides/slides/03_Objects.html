Accessing Properties

Dot notation

                    
                        const john = {
                            name: 'John',
                            age: 24
                        };

                        john.lastName = 'Hero';

                        console.log(john.age); // 24
                        console.log(john.lastName); // Hero
                        console.log(john.middleName); // undefined

Accessing Properties

Bracket Notation

                    
                        const john = {
                            name: 'John',
                            age: 24
                        };

                        john['lastName'] = 'Hero';

                        console.log(john['age']); // 24
                        console.log(john['lastName']); // Hero
                        console.log(john['middleName']; // undefined
